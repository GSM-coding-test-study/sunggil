## 문제 설명
---
정수 n이 매개변수로 주어질 때 n의 각 자리 숫자의 합을 return하도록 solution 함수를 완성해주세요

## 입출력 예
---
| n      | given |
|:--------:|:-------:|
| 1234   |  10  |
| 930211 | 16|

## 코드
---

```
class Solution {
    fun solution(n: Int): Int {
        var answer: Int = 0
        var given = n
        
        while (given != 0) {
            answer += given % 10 // 마지막 한자리 수 부터 구함
            given /= 10 // 마지막 한자리 수 삭제
        }
        
        return answer
    }
}
```
